# Fabric Library

## 使用規則

- 只有吊牌、供應商規格、正式檢驗或既有 Product DNA 已確認的資料，才可寫成商品事實。
- 名稱含「萊賽爾、冰動、涼感」不等於已確認成分或機能。
- 沒有數值時，以「未量測／待補充」記錄，不用同類布料平均值代替。
