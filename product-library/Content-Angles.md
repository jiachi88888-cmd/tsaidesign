# Content Angles

## 使用規則

- 每支影片只選一題、只講一個重點，45–60 秒且不得超過 60 秒。
- 全部內容遵守 [Brand-Language.md](Brand-Language.md)；固定鏡位順序為全身 → 腰部 → 口袋 → 布料 → 走路 → 坐下 → 轉身 → 近拍布料 → 結尾。
- 標示「需實測／需資料／需真實故事」的題目，條件未完成前不可寫成正式腳本。
- 不虛構闆娘經歷、顧客回饋、銷量、步數、旅行、洗滌結果或機能數據。
