- 狀態：唯一有效

## 目的

本系統是 Codex 唯一可使用的 TSAIDESIGN 剪輯導演知識架構。

Codex 不負責企劃或文案，只負責理解 Product DNA 與 ChatGPT 已提供的企劃／腳本，反推拍攝需求並完成影片。

## 目前保留的知識

- Fabric DNA：夏季共用哥弟針織面料、冰動牛仔四面高彈針織牛仔布。
- Waist DNA：倒梯形魔術三角。
- Fit DNA：六款現有商品的已確認版型與季節摘要。
- Product Strength：002「不起白絲、不長白毛」、005「真正的大尺碼」。
- Shot DNA：商品說法對應的必要證據畫面與 TSAIDESIGN 固定素材順序。
