# Script Rules

Codex 永遠不自行產生、重寫、補寫或優化腳本。

品牌定位、商品企劃、Hook、腳本、CTA 與行銷策略由 ChatGPT 提供。

Codex 收到腳本後，只負責逐句分析證據、反推素材、提出補拍、規劃鏡頭並完成剪輯。

所有腳本都必須依照 `04_Script-Template.md` 建立。

未來所有短影音都引用此模板。

剪輯工作依 [Creative-Editing-Director.md](../../../07_Video/Creative-Editing-Director.md) 執行。
