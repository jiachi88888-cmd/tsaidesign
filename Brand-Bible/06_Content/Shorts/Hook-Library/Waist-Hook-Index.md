# 清涼系列共用腰頭 Hook Index

## 闆娘確認範圍

以下四款商品共用倒梯形魔術三角腰頭 Hook 資料：

1. 珍瘦 V辣褲（清涼 V辣褲）。
2. 春夏清涼小直筒。
3. 春夏富貴長褲（清涼長褲）。
4. 春夏富貴七分褲（清涼七分褲）。

確認日期：2026-07-19。

兩款冰動牛仔商品尚未確認相同腰頭結構，不可套用本索引。

## 共用原因

四款商品皆已確認採用倒梯形魔術三角，搭配高彈性針織布料，用於修飾小腹、包覆肚子，減少勒肚子與緊繃感，並支援站立、坐下及久坐的舒適情境。

## 使用規則

1. 四款商品製作腰頭影片前，先讀本索引，再讀對應 Hook 的完整卡片。
2. Hook 可以共用，但畫面、商品名稱、褲長與版型說明必須換成當次實際商品。
3. 核准 Hook 原句不得自行改寫；需要更換「長褲／七分褲」等字詞時，先建立新的 `【AI 建議】` 版本交由闆娘核准。
4. 不因共用腰頭設計，就把不同商品的褲長、褲管、口袋、布料或尺寸資料互相套用。
5. 牛仔商品未經闆娘確認前，不得引用這組共用腰頭 Hook。
