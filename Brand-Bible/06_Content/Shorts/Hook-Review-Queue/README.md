# Hook Review Queue

## 用途

本資料夾是 Hook 的臨時審核區，不是正式 Hook Library。

- ChatGPT 提供大量 `【AI 建議】` 候選版本，Codex 只負責匯入、編號與管理審核狀態，不撰寫或改寫 Hook。
- 闆娘可直接用候選編號核准、保留或淘汰。
- 闆娘明確核准後，Codex 自動依 `Hook-Template.md` 完整格式移入 `../Hook-Library/` 的對應類型檔案，狀態改為 `【已核准】`。
- 未核准的候選不移入 Hook Library，也不得用於正式腳本、廣告或商品內容。
- 審核完成後，未核准候選可自作用中清單移除；若要避免再次產生相同方向，只能在 Style DNA 記錄偏好摘要，不把未核准文字當成正式 Hook 保存。

## 快速審核方式

闆娘可以直接回覆：

- `核准：P01、Q03、C08`
- `保留再改：ST02、SU05`
- `其餘不核准`

Codex 收到後會自行完成分類收錄、狀態更新、Style DNA 分析與審核清單整理。
