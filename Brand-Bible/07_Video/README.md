# TSAIDESIGN Video Editing System

## 唯一角色入口

所有 TSAIDESIGN 短影音分析、素材要求、分鏡與剪輯工作，先讀 [Creative-Editing-Director.md](Creative-Editing-Director.md)。

## 執行文件

- [Script.md](Script.md)：ChatGPT 腳本輸入與逐句證據分析。
- [Opening-Hooks.md](Opening-Hooks.md)：前三秒 Hook 證明方式。
- [Shot-List.md](Shot-List.md)：必拍素材、鏡位、景別與秒數。
- [Storyboard.md](Storyboard.md)：逐句分鏡與剪輯順序。
- [Editing.md](Editing.md)：剪輯節奏與權限邊界。
- [Subtitle.md](Subtitle.md)：字幕斷句與畫面配置。
- [BGM.md](BGM.md)：音樂方向。
- [CTA.md](CTA.md)：原 CTA 的剪輯執行。
- [Video-Checklist.md](Video-Checklist.md)：輸入、素材與成片檢查。

## 平台入口

- [Reels.md](Reels.md)：Facebook Reels／Instagram Reels。
- [TikTok.md](TikTok.md)：TikTok。
- [Shorts.md](Shorts.md)：YouTube Shorts。

平台入口共用同一角色與剪輯標準，不得因平台不同而改寫 Hook、腳本、CTA、賣點或企劃方向。
