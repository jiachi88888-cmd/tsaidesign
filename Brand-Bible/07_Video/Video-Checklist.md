# Video Checklist

## 輸入檢查

- [ ] 已收到 ChatGPT 企劃。
- [ ] 已收到 ChatGPT Hook。
- [ ] 已收到 ChatGPT 完整腳本。
- [ ] 已收到 ChatGPT CTA。
- [ ] 已確認商品與對應 Product Knowledge Library。
- [ ] 已檢查 Product Strength Index，並讀取腳本使用的 Strength 卡片。
- [ ] 沒有自行改寫或補寫內容。

## 分析與素材檢查

- [ ] 已分析影片目的。
- [ ] 已逐句列出需要證明的內容。
- [ ] 已列出 `【必拍素材】★★★★★`。
- [ ] 已列出 `【建議補拍】★★★★`。
- [ ] 已列出 `【可有可無】★★★`。
- [ ] 已標示建議鏡位、景別與拍攝秒數。
- [ ] 所有舒服、包覆、彈性、不起白絲、褲長或顯瘦說法都有規定證據畫面。
- [ ] Product Strength 的必拍素材、鏡位、景別與動作已完整列入。
- [ ] 素材不足時已提出具體補拍要求。

## 剪輯檢查

- [ ] 前三秒最快證明 Hook。
- [ ] 前三秒沒有品牌介紹、商品介紹、Logo 或大量文字。
- [ ] 平均每 0.8–2 秒切鏡，重點畫面保留 2–3 秒。
- [ ] 每一句腳本都有對應畫面。
- [ ] 字幕沒有新增或改寫賣點。
- [ ] BGM 不蓋過口白。
- [ ] Boom、Whoosh、Click 只放在有功能的位置。
- [ ] CTA 使用 ChatGPT 原文。
- [ ] 成片沒有用剪輯製造不存在的效果。

## 每次固定輸出模板

```text
【影片目的】

【需要證明的內容】

【必拍素材】
★★★★★

【建議補拍】
★★★★

【可有可無】
★★★

【建議鏡位】

【建議景別】

【建議拍攝秒數】

【建議剪輯順序】

【建議字幕】

【建議BGM】

【建議音效】
```

完整角色規則見 [Creative-Editing-Director.md](Creative-Editing-Director.md)。
